Installation guide
1
Download the repo EASY
Go to github.com/Banisher2005/SRM-Desk
Click the green Code button → Download ZIP
Extract the ZIP anywhere on your PC

2
Open Chrome Extensions CHROME
In Chrome, go to chrome://extensions
Turn on Developer mode using the toggle in the top-right corner

3
Load the extension ONE TIME
Click Load unpacked → select the extracted SRM-Desk folder
The extension will appear in your Chrome toolbar

4
Sync your data
Go to academia.srmist.edu.in and log in normally
Click the SRM Desk icon in your Chrome toolbar
Click ⚡ Sync Data from SRM and wait ~60 seconds
Click 🚀 Open Dashboard — done!
